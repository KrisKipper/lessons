Упражнения по BOM

Что такое BOM?

BOM (Browser Object Model) - это объектная модель браузера, которая позволяет JavaScript взаимодействовать с браузером. В отличие от DOM (Document Object Model), который работает с содержимым страницы, BOM предоставляет доступ к таким возможностям браузера как:

  • Управление окнами (открытие/закрытие)
  • Работа с таймерами (setTimeout, setInterval)
  • Доступ к информации о браузере (navigator)
  • Управление историей просмотров (history)
  • Работа с cookies и локальным хранилищем
  • Отображение диалоговых окон (alert, confirm, prompt)

2. Использование setTimeout

При нажатии на кнопку (#showDelayedMessage) через 3 секунды в консоли должно появиться сообщение "Прошло 3 секунды!", а текст внутри круга должен измениться на "Таймер сработал!".

Ожидаю...

Подсказка:

Используй window.setTimeout() с задержкой 3000 мс. Для изменения текста в круге - измени свойство textContent элемента.

3. Использование setInterval

При нажатии на кнопку (#startCounter) запустите таймер, который каждую секунду будет увеличивать значение счетчика на 1 и выводить его в круг. Кнопка (#stopCounter) должна останавливать таймер.

0

Подсказка:

Для запуска используй window.setInterval() с интервалом 1000 мс. Для остановки - clearInterval(). Не забудь сохранить ID интервала в переменную.

4. Работа с окнами браузера

Создайте кнопки для:

  • Открытия нового окна с адресом https://example.com
  • Закрытия текущего окна (работает только если окно было открыто скриптом)
  • Изменения размеров текущего окна

Подсказка:

  • Для открытия: window.open(url)
  • Для закрытия: window.close()
  • Для изменения размеров: window.resizeTo(width, height)
Сохрани ссылку на открытое окно в переменную, чтобы потом его закрыть.

5. Работа с location

Создайте функционал для:

  • Показа текущего URL в консоли
  • Перенаправления на https://google.com
  • Перезагрузки текущей страницы

Подсказка:

  • Текущий URL: window.location.href
  • Перенаправление: window.location.href = url
  • Перезагрузка: window.location.reload()

6. Работа с confirm и prompt

Создайте кнопки для:

  • Отображения confirm с вопросом "Вы уверены?" и выводом результата в консоль
  • Отображения prompt с вопросом "Как вас зовут?" и выводом результата в alert

Подсказка:

  • confirm(message) возвращает true/false
  • prompt(message) возвращает введенную строку или null
Результаты выводи в консоль через console.log() или в alert через window.alert()
