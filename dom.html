Упражнения по JavaScript

1. Смена классов

При нажатии на кнопку (#changeColor) класс _theme_pink должен удалиться и установиться класс _theme_blue. После чего к кнопке добавить класс _disabled

При нажатии на кнопку (#toggleColor) класс _theme_pink должен переключаться на _theme_blue и при повторном клике обратно

2. Удаление элементов

При нажатии на кнопку (#deleteBtn) удалить все круги в этой области

3. Работа с атрибутами

При нажатии на кнопку (#addAttr) у всех кружков в этой области должен добавляться атрибут `data-active="true"`.

При нажатии на кнопку (#removeAttr) у всех кружков в этой области должен удалиться атрибут `data-active`.

4. Изменение текста

При нажатии на кнопку (#changeText) текст внутри кружка должен измениться на "Я изменён!"

Я изменюсь!

5. Скрытие элементов

При нажатии на кнопку (#hideCircles) все кружки в этой области должны скрыться (используй `display: none`)

6. Создание/ удаление элементов

При нажатии на кнопку (#createFromInput) создаётся новый круг с текстом, введённым в поле.

Привет!

7. Использование alert

При нажатии на кнопку (#showAlertAndLog) отобразить сообщение с текстом "Это alert!" (используй alert)

8. Использование setTimeout

При нажатии на кнопку (#showDelayedMessage) через 3 секунды в консоли должно появиться сообщение "Прошло 3 секунды!", а текст внутри круга должен измениться на "Таймер сработал!".

Ожидаю...

9. Перемещение элемента

При нажатии на кнопку (#moveCircle) круг должен переместиться из левой области блока .s-exercises__content в правую и наоборот.

10. Работа с методами массива

У тебя есть массив чисел: `[1, 2, 3, 4, 5]`. При нажатии на кнопку (#processArray) должны выполниться следующие действия:

  • Все числа должны быть увеличены на 1.
  • Отфильтровать только чётные числа.
  • Получившийся массив должен быть выведен в консоль.

11. Работа с методами строк

У тебя есть строка: `"javascript is fun"`. При нажатии на кнопку (#processString) должны быть выполнены следующие действия:

  • Преобразовать строку в верхний регистр.
  • Заменить все пробелы на подчеркивания ("_").
  • Результат должен быть выведен в консоль.
