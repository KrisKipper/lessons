Упражнения по DOM

1. Смена классов

При нажатии на кнопку (#changeColor) класс _theme_pink должен удалиться и установиться класс _theme_blue. После чего к кнопке добавить класс _disabled

1.1. Смена классов

При нажатии на кнопку (#toggleColor) класс _theme_pink должен переключаться на _theme_blue и при повторном клике обратно

2. Удаление элементов

При нажатии на кнопку (#deleteBtn) удалить все круги в этой области

3. Работа с атрибутами

При нажатии на кнопку (#addAttr) у всех кружков в этой области должен добавляться атрибут `data-active="true"`.

При нажатии на кнопку (#removeAttr) у всех кружков в этой области должен удалиться атрибут `data-active`.

4. Изменение текста

При нажатии на кнопку (#changeText) текст внутри кружка должен измениться на "Я изменён!"

Я изменюсь!

5. Скрытие элементов

При нажатии на кнопку (#hideCircles) все кружки в этой области должны скрыться (используй `display: none`)

6. Создание/ удаление элементов

При нажатии на кнопку (#createFromInput) создаётся новый круг с текстом, введённым в поле.

Привет!

7. Наведение мыши

При наведении на круг (#hoverCircle) он должен менять цвет на синий, при уходе — обратно на розовый.

Наведи на меня

Подсказка:

Используй события mouseenter и mouseleave

8. Клавиатура

Нажатие Enter создаёт новый круг с текстом из поля ввода. Escape очищает поле.

9. Фокус на инпут

При фокусе на поле появляется сообщение. При уходе фокуса — исчезает.

10. Двойной клик

Двойной клик по кругу (#dblCircle) увеличивает его, повторный — уменьшает.

Двойной клик

11. Перетаскивание

Круг (#dragCircle) можно перетаскивать мышью по экрану.

Тяни

12. Мигающий круг

При наведении на круг, он начинает мигать. При уходе — перестаёт.

Мигай
