Примеры решений JavaScript-задач

1. Работа с классами

Задача: При клике на кнопку добавить/удалить класс у элемента.

Круг

Решение:

// Получаем элементы
const circle = document.getElementById('targetCircle');
const addBtn = document.getElementById('addClassBtn');
const removeBtn = document.getElementById('removeClassBtn');
const toggleBtn = document.getElementById('toggleClassBtn');

// Добавляем обработчики событий
addBtn.addEventListener('click', () => {
    // Добавляем класс
    circle.classList.add('highlight');
    circle.textContent = 'Класс добавлен';
});

removeBtn.addEventListener('click', () => {
    // Удаляем класс
    circle.classList.remove('highlight');
    circle.textContent = 'Класс удалён';
});

toggleBtn.addEventListener('click', () => {
    // Переключаем класс
    circle.classList.toggle('highlight');
    circle.textContent = circle.classList.contains('highlight')
        ? 'Класс есть'
        : 'Класса нет';
});

Пояснение: Используем методы classList.add(), classList.remove() и classList.toggle() для управления классами элемента.

2. Создание и удаление элементов

Задача: Создавать новые элементы при клике и удалять их.

Решение:

const container = document.getElementById('elementsContainer');
const createBtn = document.getElementById('createElementBtn');
const clearBtn = document.getElementById('clearElementsBtn');

createBtn.addEventListener('click', () => {
    // Создаём новый элемент
    const newElement = document.createElement('div');
    newElement.className = 'circle';
    newElement.textContent = `Элемент ${container.children.length + 1}`;

    // Добавляем в контейнер
    container.appendChild(newElement);
});

clearBtn.addEventListener('click', () => {
    // Удаляем все дочерние элементы
    while (container.firstChild) {
        container.removeChild(container.firstChild);
    }
});

Пояснение: Используем createElement() для создания и appendChild() для добавления элементов. Для удаления - removeChild().

3. Работа с атрибутами

Задача: Добавлять и удалять атрибуты у элемента.

Проверь атрибуты

Решение:

const attrCircle = document.getElementById('attrCircle');
const setAttrBtn = document.getElementById('setAttrBtn');
const removeAttrBtn = document.getElementById('removeAttrBtn');

setAttrBtn.addEventListener('click', () => {
    // Устанавливаем атрибут
    attrCircle.setAttribute('data-info', 'Это важные данные');
    attrCircle.textContent = 'Атрибут установлен';
});

removeAttrBtn.addEventListener('click', () => {
    // Удаляем атрибут
    attrCircle.removeAttribute('data-info');
    attrCircle.textContent = 'Атрибут удалён';
});

Пояснение: Используем setAttribute() и removeAttribute() для управления атрибутами элемента.

4. Перемещение элементов

Задача: Перемещать элемент между двумя областями.

Перемести меня

Решение:

const movableCircle = document.getElementById('movableCircle');
const leftArea = document.getElementById('leftArea');
const rightArea = document.getElementById('rightArea');
const moveBtn = document.getElementById('moveElementBtn');

moveBtn.addEventListener('click', () => {
    // Проверяем, где сейчас находится элемент
    if (leftArea.contains(movableCircle)) {
        // Перемещаем в правую область
        rightArea.appendChild(movableCircle);
    } else {
        // Перемещаем в левую область
        leftArea.appendChild(movableCircle);
    }
});

Пояснение: Используем contains() для проверки местоположения элемента и appendChild() для перемещения.

5. Таймеры и задержки

Задача: Выполнить действие с задержкой.

Ожидание...

Решение:

const timerCircle = document.getElementById('timerCircle');
const startBtn = document.getElementById('startTimerBtn');
const stopBtn = document.getElementById('stopTimerBtn');
let timerId;

startBtn.addEventListener('click', () => {
    timerCircle.textContent = 'Таймер запущен...';

    // Устанавливаем таймер на 3 секунды
    timerId = setTimeout(() => {
        timerCircle.textContent = 'Таймер сработал!';
    }, 3000);
});

stopBtn.addEventListener('click', () => {
    // Останавливаем таймер
    clearTimeout(timerId);
    timerCircle.textContent = 'Таймер остановлен';
});

Пояснение: Используем setTimeout() для выполнения кода с задержкой и clearTimeout() для отмены.

6. Работа с массивами

Задача: Преобразовать массив чисел.

Решение:

const processBtn = document.getElementById('processArrayBtn');
const outputDiv = document.getElementById('arrayOutput');

processBtn.addEventListener('click', () => {
    const numbers = [1, 2, 3, 4, 5];

    // Увеличиваем каждое число на 1
    const incremented = numbers.map(num => num + 1);

    // Фильтруем чётные числа
    const evenNumbers = incremented.filter(num => num % 2 === 0);

    // Выводим результат
    outputDiv.innerHTML = `
        Исходный массив: [${numbers.join(', ')}]

        После увеличения: [${incremented.join(', ')}]

        Чётные числа: [${evenNumbers.join(', ')}]
    `;

    // Также выводим в консоль
    console.log('Результат обработки массива:', evenNumbers);
});

Пояснение: Используем методы map() для преобразования и filter() для фильтрации массива.
